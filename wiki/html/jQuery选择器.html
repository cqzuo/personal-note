选择器

基本

  1. 4中基本类型
    • id
      • #
      • 根据给定的ID匹配一个元素
      • 例子 
         
        <p>id="notMe"</p>
        
 
        id="myDiv"
        
 $("#myDiv");
    • element
      • 根据给定的元素名匹配所有元素
      • 例子 
        DIV1
        

        DIV2
        
<span>SPAN</span>
$("div");
    • class
      • .
      • 根据给定的类匹配元素
      • 例子 
        div class="notMe"
        

        div class="myClass"
        
<span class="myClass">span class="myClass"</span>
$(".myClass");
    • 任意
      • 匹配所有元素

层级

  1. ancester desendant
    • 在给定的祖先元素下匹配所有的后代元素
    • $("form input")
  2. parent>child
    • 在给定的父元素下匹配所有的子元素
    • $("form > input")
  3. prev+next
    • 匹配所有紧接在 prev 元素后的 next 元素
    • $("label + input")
  4. prev~siblings
    • 匹配 prev 元素之后的所有 siblings 元素
    • $("form ~ input")

内容

  1. :first
    • 第一个元素
  2. :last
    • 最有一个元素
  3. :not
    • 去除所有与给定选择器匹配的元素
    • 例子:获取未选中的元素 
      <input name="apple" />
<input name="flower" checked="checked" />
$("input:not(:checked)")
  4. :even
    • 索引为偶数
  5. :odd
    • 索引为基数
  6. :eq
    • 匹配索引
  7. :gt
    • 大于
  8. :lt
    • 小于
  9. :header
    • 标题元素
  10. :animated
    • 正在执行的动画效果
    • 例子 对未执行的动画施加一个特效 
      <button id="run">Run</button>
      
 $("#run").click(function(){
 $("div:not(:animated)").animate({ left: "+=20" }, 1000);
});

内容匹配

  1. :contains
    • 匹配给定文本的元素
    • 例子 匹配含有"John"的div 
      John Resig
      

      George Martin
      

      Malcom John Sinclair
      

      J. Ohn
$("div:contains('John')")
  2. :empty
    • 不含有子元素或文本元素
  3. :has
    • 含有指定元素的元素
  4. :parent
    • 含有子元素或文本元素

可见性

  1. :hiden
  2. :visible

属性

  1. [attribute]
    • 匹配含有该属性的元素
  2. [attribute=value]
    • 属性为指定值
  3. [atu!=value]
    • 属性为指定值
  4. [attribute^=value]
    • 开始
  5. [attribute$=value]
    • 结尾
  6. [attribute*=value]
    • 任意包含
  7. [selector1][selector2]...
    • 同时满足多个条件

子元素

  1. :nth-child
    • 匹配指定元素个数
  2. :first-child
    • 第一个元素
  3. :last-child
    • 最后一个元素
  4. :only-child
    • 唯一匹配值
